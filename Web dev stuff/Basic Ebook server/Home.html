Simple Ebook server


Welcome to Simple Ebook server

An accessible place to store all your Ebooks on the network

How to use the server

Browse library

The library neatly displays all the books you have saved on the server, located in the /Books folder.

Your books are shown in their folders to make them easier to find and can be be managed using the library and book options on the management page.

Read

You can read your books on the browser using our library page, just look for a green file type after the name and click to enjoy.

Books that are in an unsupported file type will show up as yellow(WIP/beta support) or red (currently unsupported), you can still read them, but they will have to be downloaded first.

Manage

The management page allows you to manage your library, it's functions include uploading, deleting and renaming books and folders, creating folders and moving books between folders.

It also includes server options like disabling downloads, blacklisting and disabling readers, to see all the functions please visit the management page.

The server is located at: http://192.168.0.110/Simple_Ebook_Server/Home.html

The books folder is located at: web root/Simple_Ebook_server/Books



Currently supported formats

We currently support online reading of the following formats, other formats will need to be download (if enabled) to read.
.pdf
.txt(beta)
.epub(beta)
