Welcome to simple EBook server

An accessible place to store all your Ebooks on the network



Please visit the "Library" tab to browse your library






Dev stuff:

Since this system is actively under development you may find that some of your files are out of date
Please check the repo to get the latest versions.
You should be able to safely replace exisiting versions with the new ones by overwriting them.
Check the date modified indicators to see which files are newer.
